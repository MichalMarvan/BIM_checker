# Code Review

Proveď code review aktuálních změn. Zkontroluj:

## Checklist
- [ ] Bezpečnost (injekce, XSS, CSRF, OWASP top 10)
- [ ] Error handling a edge cases
- [ ] Výkon (N+1 queries, zbytečné renderování, memory leaks)
- [ ] Čitelnost a konvence pojmenování
- [ ] DRY – žádná zbytečná duplikace
- [ ] Testy – pokrývají happy path i error cases?
- [ ] Žádné zakomentované bloky nebo debug printy

## Výstup
Shrň nálezy jako seznam: kritické → důležité → návrhy.
